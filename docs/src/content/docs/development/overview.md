Welcome to the developer documentation for Converse!

Here you'll learn how to add new features, extend functionality through plugins,
and create your own customized version of Converse.

Converse is a community project and largely volunteer-driven.

We're grateful for contributions of all kinds, so please don't hesitate to
make a [GitHub pull request](https://help.github.com/categories/collaborating-with-issues-and-pull-requests/)
to fix a bug or add new functionality. Whether it's your first open source
contribution or your hundredth, we'd love to have you involved!
